How to use
------------

**Windows:**
```
cd ../GlobalXNameSort/bin/Release/
name-sorter ./unsorted-names-list.txt
```

**Mac:**
```
cd ../GlobalXNameSort/bin/Release/
mono name-sorter.exe ./unsorted-names-list.txt
```
